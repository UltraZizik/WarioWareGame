# Dogwario 🐾

It is warioWare like game and that means that you have to complete a game before the timer runs out!

# 🎮 Game Modes
This project features two fast, dog-themed minigames:

## Game 1: Platformer! (Micro-Platformer) 🦴

The Objective: Get all the apples you can!

Controls: AD and spacebar or arrow keys to run and jump.

## Game 2: Clicker! (Speed-Clicker) 🐕

The Objective: Click on four apples!

Controls: Left Mouse Click.

# 🏆 Win & Loss States


### The Win Screen 🎉: Success! You beat my microgames and unlocked eternal happines.

### The Losing Screen 🌧️: Too slow! Don't be scared to try again once more!
